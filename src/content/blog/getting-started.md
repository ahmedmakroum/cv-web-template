# Getting Started with This Portfolio Template

Welcome! This is an example blog post. You can replace this with your own content.

## How to Use This Template

1. **Personalize Your Information**: Update all placeholder text with your actual details
2. **Add Your Projects**: Create new Markdown files in `src/content/projects/`
3. **Write Blog Posts**: Share your thoughts in `src/content/blog/`
4. **Customize Styling**: Modify colors and styles in `tailwind.config.js`
